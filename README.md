# 💻 Movie-Zone


* Next.js + graphql + amplify 를 이용한 간이영화예매 및 게시판 기능 애플리케이션 구현

## 주요 기능

* 로그인/회원가입 기능
* 영화 api 이용한 리스트 표현
* 게시판 기능

## 주요 기술

* Next.js
* TypeScript
* styled-components
* Recoil
* aws amplify
* graphql
